关于

Google Chrome，又称Google浏览器，是一个由Google(谷歌)公司开发的网页浏览器。

Mozilla Firefox，中文俗称"火狐"(正式缩写为Fx或fx，非正式缩写为FF)，是一个自由及开放源代码网页浏览器。

IE，是美国微软公司（Microsoft）推出的一款网页浏览器。

Safari,苹果计算机的最新操作系统Mac OS X中新的浏览器，用来取代之前的Internet Explorerfor Mac。
